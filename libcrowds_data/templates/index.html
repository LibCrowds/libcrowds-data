Data

This page contains all of the open datasets generated via from the experimental crowdsourcing projects hosted on this platform. Everything on this page is made available under CC0, meaning that it can be used by anyone without any copyright restrictions. We are very keen for these materials to be used in innovative ways to further research into new technologies. For instance, the digitised images used as the initial project input, alongside the results transcribed or derived by other other means might prove useful for testing pattern recognition applications such as multi-lingual and hand-writing OCR technologies.

The digitised images used in our crowdsourcing projects can generally be downloaded as datasets via data.bl.uk.

Get in touch via {% if "pybossa_discourse" in plugins %} Visit the the LibCrowds Community {% else %} email {% endif %} to let us know how you have made use of the data, or if you have any further enquiries.
